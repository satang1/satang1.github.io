Galaxies was an academic project I created in Java. It is based off of one of [Simon Tatham's](https://www.chiark.greenend.org.uk/~sgtatham/puzzles/) GUI games. You can play a demo version on his website [here](https://www.chiark.greenend.org.uk/~sgtatham/puzzles/js/galaxies.html). The goal of the game is to bound each "center" dot into symmetrical regions (or galaxies). And each region must only contain one center.
